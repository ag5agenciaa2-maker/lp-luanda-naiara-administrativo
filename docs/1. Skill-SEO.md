## Execution Workflow (Standard Operating Procedure)

When asked to perform a full SEO/GEO audit, follow these steps:
1. **Discovery**: Use file reading tools (`Glob`, `Grep`, `Read`) to find `.html`, `.css`, and `.js` files. Read the `<head>` and `<body>` structures.
2. **Technical Check**: Verify meta tags, Open Graph, Twitter cards, Schema.org (JSON-LD), robots.txt, and sitemap.xml.
3. **Performance Check**: Analyze `<img>` tags (formats, `loading="lazy"`, `fetchpriority`, `alt` attributes) and scripts (`defer` / `async`).
4. **GEO & EEAT Check**: Look for localized content (NAP - Name, Address, Phone), FAQs, timelines ("updated at"), and author/company credentials.
5. **Implementation**: Modify the files to apply the fixes directly.
6. **External Reporting**: Always generate a final markdown artifact reporting what was changed and **listing the external/off-page tasks** that you couldn't do.

## Core Web Vitals & Technical Best Practices

| Metric | Good | Poor | How to Check / Fix in Code |
|--------|------|------|----------------------------|
| **LCP** | < 2.5s | > 4.0s | Hero image needs `fetchpriority="high"`. Preconnect to font domains. |
| **INP** | < 200ms | > 500ms | Defer heavy JS (`<script defer>`). |
| **CLS** | < 0.1 | > 0.25 | Images must have explicit `width` and `height`. |

**Additional Tech Checks:**
- Use modern image formats (`.webp` or `.avif`).
- Use `loading="lazy"` on all images outside the initial viewport (below the fold).
- Ensure fonts are loaded with `font-display: swap`.

## Comprehensive SEO/GEO Checklists

### Technical & Local SEO Checklist
- [ ] Language (`lang="pt-BR"`) and Charset (`UTF-8`) defined
- [ ] `robots.txt` configured correctly (Allow AI bots if GEO is desired)
- [ ] `sitemap.xml` generated and correct
- [ ] Canonical tags (`<link rel="canonical">`) present
- [ ] Mobile viewport meta tag present
- [ ] Local SEO (Geo tags: `geo.region`, `geo.position`, `geo.placename`)
- [ ] NAP info (Name, Address, Phone) visible in HTML text (not just images)

### Social & Semantic Checklist
- [ ] Open Graph (`og:title`, `og:description`, `og:image`, `og:url`, `og:type`)
- [ ] Twitter Cards (`twitter:card`, `twitter:title`, `twitter:image`)
- [ ] Accessibility: `<main>`, `<nav>`, `<footer>` structured correctly
- [ ] Accessibility: `aria-label` on icon-only buttons

### Content SEO Checklist
- [ ] **Title tags optimized (Front-loading Strategy):** (50-60 chars). ALWAYS place the main keyword/category and location at the very beginning of the title to increase CTR, followed by the brand name. (e.g., "Advogado em Campo Grande RJ | Nome do Escritório").
- [ ] Meta descriptions (150-160 chars) - actionable and engaging
- [ ] H1-H6 hierarchy correct (only one H1 per page)
- [ ] Internal linking structure using descriptive anchor texts
- [ ] Descriptive image `alt` texts (avoid generic "image1")

### Essential Schema Markups (JSON-LD)
Depending on the project, implement:
1. **Organization** or **LocalBusiness** (AutoRepair, MedicalClinic, etc.)
2. **FAQPage** (crucial for GEO and Rich Snippets)
3. **AggregateRating** (for testimonials and reviews)
4. **Article** (for blogs/news)

### GEO Checklist
- [ ] FAQ sections present (direct answers to common questions)
- [ ] Author/Company credentials visible
- [ ] Statistics with sources clearly stated
- [ ] Clear, extractable definitions
- [ ] Expert quotes attributed
- [ ] "Last updated" timestamps on content

## External/Off-Page Dependencies (Unfixables)

> **CRITICAL:** As an AI modifying code, you cannot fix external SEO factors. Always include a checklist of these external dependencies in your final report for the user to handle manually.

**External Audit Checklist to provide the user:**
- [ ] **Google My Business (GMB)**: Verify and complete the local profile, ensure NAP matches the website.
- [ ] **Google Search Console**: Submit the `sitemap.xml` and request indexing.
- [ ] **Google Analytics / Tag Manager**: Set up tracking codes if not present.
- [ ] **Backlink Building**: Acquire high-quality backlinks from local directories or industry partners.
- [ ] **Social Media Profiles**: Link back to the website from Instagram, Facebook, LinkedIn, etc.
- [ ] **PageSpeed Insights Test**: Recommend the user run a live test once deployed to catch server-side delays.
- [ ] **Hosting Security**: Ensure the server forces HTTPS (SSL certificate).
