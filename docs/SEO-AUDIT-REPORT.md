# Relatório de Auditoria e Otimização SEO/GEO
**Projeto:** Luanda Naiara Advocacia  
**Data:** 24/03/2026

## 1. Melhorias Técnicas Implementadas

### 1.1 Dados Estruturados (Schema.org)
Implementamos marcação JSON-LD em todas as páginas para facilitar a compreensão do conteúdo por motores de busca e IAs:
- **Index:** `LegalService` com o nome oficial do GMB **"Luanda Naiara - Advogada (Concurso Público)"**, `AggregateRating` (309 avaliações) e `FAQPage`.
- **Sobre:** `Person` (perfil da Dra. Luanda) e `AboutPage`, vinculando à organização oficial.
- **Redes Sociais:** Atualizado `sameAs` com links corrigidos para GMB, Facebook, Instagram e Threads.

### 1.2 Performance e Core Web Vitals
Otimizamos o carregamento de recursos críticos:
- **LCP (Largest Contentful Paint):** Adicionado `fetchpriority="high"` na imagem do hero e removido o atraso de opacidade inicial para carregamento imediato.
- **Lazy Loading:** Implementado `loading="lazy"` em todas as imagens abaixo da dobra para reduzir o tempo de carregamento inicial.
- **Acessibilidade:** Atualização de tags `alt` para descrição SEO e acessibilidade.

### 1.3 Metadados e GEO (Generative Engine Optimization)
- Refinamento de metatags para nichos específicos (nomeação e posse, concurso público).
- **Robots.txt:** Atualizado para permitir e orientar bots de IA (GPTBot, Claude-Web, PerplexityBot, etc.), essencial para visibilidade em buscas generativas.
